### 4.2 Tax ID / EIN

**EIN = Employer Identification Number (IRS)**

**Check if you have EIN:**
1. Did you file taxes last year? (If sole proprietor, might not have separate EIN)
2. Did you apply for one? Check email for **"IRS confirmation letter"**

**If you DON'T have EIN:**
1. Go to: https://www.irs.gov/ein
2. Fill form: **SS-4 Application for EIN**
3. Can apply online (instant) or by phone
4. You get EIN immediately
5. Print or save confirmation

**File preparation:**
- Rename: **IRS_EIN_Confirmation.pdf**
- Save to: `/tax/`

**Must show:**
```
‚úÖ Your name or business name
‚úÖ EIN Number: ##-#######
‚úÖ Confirmation/approval date
‚úÖ IRS letterhead
```

### 4.3 Certificate of Incorporation (LLC Only)

**ONLY if you created an LLC (legal entity):**

1. Go to: https://www.sos.ca.gov
2. Search business database
3. Download **"Certificate of Good Standing"** or **"Articles of Incorporation"**
4. Save: **Certificate_Incorporation_[State].pdf**

- Skip this document (not needed)

## PART 7: COMMON LSA REJECTION REASONS & FIXES

### Reason 1: Insurance Certificate Expired

```
ERROR MESSAGE:
"Insurance certificate must be valid for 12+ months"

FIX:
1. Renew insurance
2. Get new certificate
3. Re-upload to application
4. Resubmit
```

### Reason 2: Business License Missing or Invalid

```
ERROR MESSAGE:
"Business license number not found in LA database"

FIX:
1. Verify license exists (call LA Dept of Finance)
2. Make sure business name matches EXACTLY
3. Re-upload clearer photo
4. Resubmit
```

### Reason 3: ID Document Quality

```
ERROR MESSAGE:
"ID image is too blurry or text not readable"

FIX:
1. Re-scan ID in better lighting
2. Make sure text is 100% readable
3. Crop to show only ID (not fingers/background)
4. Re-upload
```

### Reason 4: Service Scope Mismatch

```
ERROR MESSAGE:
"You're offering services that require licensing"

FIX:
1. Review PHASE-B-LEGAL-GUARDRAIL.md
2. Ensure services are <$1,000 labor only
3. Add disclaimer: "Cosmetic repairs only, no permits"
4. Resubmit with updated scope
```

### Reason 5: EIN Not Valid

```
ERROR MESSAGE:
"EIN does not match business records"

FIX:
1. Double-check EIN is correct (no typos)
2. Verify you're using same business name as EIN application
3. Upload IRS confirmation letter
4. Resubmit
```
